# AutoDrive - Inventario de Carros

Para hacerlo funcionar:

1. Ve a la carpeta `backend`, dale `npm install` y luego `npm run dev`.
2. Haz lo mismo en la carpeta `frontend`: `npm install` y `npm run dev`.
